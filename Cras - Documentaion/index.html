React Installation

Please follow the instructions in the video to see how you can install react on your hosting:

  1. For local host -
    1. Open you command prompt
    2. npm install or npm install --legacy-peer-deps
    3. npm run dev (will start the dev server at http://localhost:5173)


  1. For locally build -
    1. Open you command prompt
    2. npm run build (will build locally and generate out folder in root directory)
  2. Build and Deploying to Vercel - https://www.youtube.com/watch?v=7T4w0QJtL-o
